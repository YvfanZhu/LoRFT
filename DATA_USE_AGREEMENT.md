# LoRFT Data Use Agreement

LoRFT is released for approved research and benchmark evaluation purposes. The dataset contains fixed highway surveillance videos, manually verified vehicle trajectories, vehicle bounding boxes, road geometry annotations, scene level splits, and evaluation scripts.

By requesting access to LoRFT, the user agrees to the following terms.

## Permitted Use

The dataset may be used only for academic research, education, and benchmark evaluation related to traffic data mining, vehicle trajectory reconstruction, computer vision, intelligent transportation systems, and related fields.

## Prohibited Use

The user may not use LoRFT for commercial deployment, law enforcement, surveillance targeting, vehicle identification, personal identification, license plate recognition, camera localization, or any attempt to infer sensitive infrastructure information.

The user may not attempt to identify individuals, vehicles, license plates, camera locations, or other sensitive information from the videos or annotations.

## Redistribution

The user may not redistribute, sublicense, publish, sell, transfer, or share the raw videos, annotations, derived copies, download links, or access credentials with any third party.

Access is granted only to the approved applicant and the research group described in the access request.

## Data Security

The user must store the dataset securely and take reasonable measures to prevent unauthorized access, copying, or redistribution.

If the data are no longer needed, the user should delete local copies.

## Citation

Any publication, report, benchmark result, or released code using LoRFT must cite the LoRFT paper and acknowledge the dataset.

## Access Revocation

The dataset maintainers reserve the right to deny, suspend, or revoke access if the user violates this agreement or provides incomplete or inaccurate access request information.

## Contact

For questions about data access or permitted use, please contact the LoRFT maintainers through the dataset repository.
