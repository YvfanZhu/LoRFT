# LoRFT Data Use Agreement

## How to Complete This Agreement

To request access to the LoRFT video data, please complete the following steps:

1. Read this Data Use Agreement carefully.
2. Copy the full text of this agreement into a document editor such as Microsoft Word, Google Docs, or Overleaf.
3. Fill in the applicant information section at the end of this agreement.
4. Add your signature and the signing date.
5. Export the completed agreement as a PDF file.
6. Upload the signed PDF to a cloud storage service with view permission enabled.
7. Submit the shared PDF link in the Hugging Face access request form under `Signed agreement link`.

Incomplete agreements, unsigned agreements, inaccessible links, or requests submitted with inconsistent information may be rejected.

## Dataset Scope

LoRFT includes public project materials and controlled access video data.

The public project materials, including paper description, public map files, public data files, code, and documentation, are maintained in the official GitHub repository:

```text
https://github.com/YvfanZhu/LoRFT
```

The controlled access video data is hosted through the LoRFT Hugging Face dataset repository:

```text
https://huggingface.co/datasets/YvfanZhu/LoRFT
```

This Data Use Agreement governs access to and use of the controlled access LoRFT video data. It does not change the public availability of the materials that are already released through the official GitHub repository.

## Definitions

For the purposes of this agreement:

1. `LoRFT video data` means the controlled access video files made available through the LoRFT Hugging Face dataset repository.
2. `User` means the individual applicant who requests and receives access to the LoRFT video data.
3. `Maintainers` means the LoRFT dataset maintainers who manage access to the video data.
4. `Approved use` means the research, education, or benchmark evaluation purpose described in the approved access request.
5. `Third party` means any person or entity other than the approved individual user.

## Access Conditions

Access to the LoRFT video data is granted only to the approved individual user.

Other members of the same research group, laboratory, institution, or organization must submit separate access requests before accessing the LoRFT video data.

The user may not share their Hugging Face account, access token, private download links, downloaded video files, or derived copies with any other person.

The user agrees that all information submitted in the Hugging Face access request and in this agreement is accurate and complete.

The maintainers may deny, suspend, or revoke access if the request is incomplete, inaccurate, inconsistent, or incompatible with this agreement.

## Permitted Use

The LoRFT video data may be used only for approved research, education, and benchmark evaluation purposes.

Permitted research areas include:

1. Traffic data mining
2. Vehicle trajectory reconstruction
3. Computer vision
4. Intelligent transportation systems
5. Spatio temporal modeling
6. Benchmark evaluation and reproducibility studies
7. Closely related academic or educational research

The user may use the LoRFT video data together with the public map files, public data files, code, and documentation provided in the official GitHub repository.

## Prohibited Use

The user may not use the LoRFT video data for:

1. Commercial deployment
2. Law enforcement
3. Surveillance targeting
4. Vehicle identification
5. Personal identification
6. License plate recognition
7. Camera localization
8. Re identification of individuals, vehicles, locations, or infrastructure
9. Inferring sensitive infrastructure information
10. Training or deploying systems for tracking, profiling, enforcement, or operational surveillance
11. Any purpose outside the approved use described in the access request

The user may not attempt to identify individuals, vehicles, license plates, camera locations, or other sensitive information from the videos or related materials.

## Redistribution and Sharing

The user may not redistribute, sublicense, publish, sell, transfer, or share the LoRFT video data with any third party.

The user may not share:

1. Raw video files
2. Edited or re encoded video files
3. Cropped video clips
4. Screenshots or frames from the videos, unless explicitly permitted by the maintainers in writing
5. Derived copies that would allow reconstruction of the original video data
6. Private download links
7. Hugging Face access tokens
8. Account credentials
9. Any mirror, archive, or duplicate copy of the controlled access video data

The user may not upload the LoRFT video data to public websites, public repositories, public cloud folders, file sharing services, or any platform that would allow unauthorized access.

## Derived Data and Outputs

The user may create internal derived data, intermediate files, features, model outputs, or evaluation results as necessary for the approved use.

The user may publish aggregate results, benchmark metrics, trained model descriptions, and research findings, provided that such publication does not expose raw video data, identifiable visual content, private download links, or sensitive location information.

Any public release of examples, figures, screenshots, video clips, or qualitative visualizations derived from the LoRFT video data requires prior written permission from the maintainers.

## Data Security

The user must store the LoRFT video data securely and take reasonable measures to prevent unauthorized access, copying, leakage, or redistribution.

The user should use appropriate technical and organizational safeguards, such as access controlled storage, secure workstations, encrypted storage when appropriate, and restricted access within the approved research environment.

The user may not store the LoRFT video data in publicly accessible cloud folders or repositories.

If the user becomes aware of unauthorized access, loss, leakage, or redistribution of the LoRFT video data, the user must notify the maintainers as soon as reasonably possible.

## Data Retention and Deletion

The user may retain the LoRFT video data only for as long as necessary for the approved use.

If the LoRFT video data is no longer needed, or if access is revoked, the user must delete local copies and any unauthorized copies under the user's control.

Upon request from the maintainers, the user should confirm deletion of the LoRFT video data.

## Citation and Acknowledgment

Any publication, report, benchmark result, released code, or public presentation using the LoRFT video data must cite the LoRFT paper and acknowledge the LoRFT dataset.

Citation information should follow the instructions provided in the official GitHub repository.

## No Warranty

The LoRFT video data is provided for research, education, and benchmark evaluation purposes.

The maintainers provide the LoRFT video data as is, without warranties of completeness, accuracy, fitness for a particular purpose, or uninterrupted availability.

The user is responsible for determining whether the LoRFT video data is suitable for their intended research or evaluation purpose.

## Compliance

The user is responsible for complying with applicable institutional policies, research ethics requirements, data management requirements, and laws relevant to their use of the LoRFT video data.

The user must not use the LoRFT video data in a way that violates this agreement, applicable policies, or applicable law.

## Access Revocation

The maintainers reserve the right to deny, suspend, or revoke access to the LoRFT video data at any time if:

1. The access request is incomplete or inaccurate
2. The user violates this agreement
3. The user attempts to redistribute or expose the video data
4. The user uses the video data outside the approved use
5. Continued access creates legal, ethical, security, or privacy concerns
6. The maintainers determine that access should no longer be provided

Revocation of access does not remove the user's obligation to comply with the restrictions in this agreement.

## Contact

For questions about data access, permitted use, citation, or this agreement, please contact the LoRFT maintainers through the LoRFT Hugging Face dataset repository or the official GitHub repository.
